ObjConv v1.04

ObjConv converts .m3 files used by Starcraft II to the OBJ model format.

Supports:
- Vertices
- Normals
- UV coords
- Faces
- Geosets

Usage:
ObjConv <path>

PS: If <path> is a directory, the application will perform a batch-conversion
    of all the .m3 files in the directory.
